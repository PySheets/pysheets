What is PySheets?

PySheets is a spreadsheet UI for Python, allowing you to import Excel sheets or copy cells from Google Sheets, perform exploratory data science, use Pandas and create charts with Matplotlib, and create reports.

PySheets is open-source and available on Github. Give us a star! You can also install PySheets safely and easily on-prem using pip install pysheets-app.

PySheets is local-first. All data is stored in the browser using IndexedDB. All your Python logic runs in the browser as well. PySheets is written in Python and runs inside the browser using PyScript.

PySheets has AI support to generate code. With PySheets, you can load any Python package and JS module to leverage the huge data-science ecosystem without needing to write much code.


Powerful Features

  • Import and export Excel sheets quickly
  • Explore and analyse data with Pandas
  • Create Python code using AI prompts.
  • Visualize your data with Matplotlib to use elsewhere.
  • Get immediate results with minimal coding.
  • Local-first, sheets are stored in your browser.
  • No kernels, so no expensive cloud costs.
  • Leverage the entire Python and JS ecosystems.
